从环境搭建开始我的 智能合约 探索。

一、安装Geth

以太坊社区提供了多种语言实现的客户端和开发库，官方推荐使用 Go-ethererum，也就是 Geth 。

Geth 官网下载链接 官网有一点小问题，通过 F12 找到了 Windows 的对应下载链接 v1.9.15

源码在 Github - Geth - Release

安装到路径 C:\workspace\geth\

命令行里面输入 geth version，正常反馈信息即为安装成功。
